# **README for slib++** #
***


## **What's slib++?** ##
 * It's a C++ version of slib.
 * It's a free and open source program based FreeBSD, everybody can use it.



## **Building slib++** ##
 * **windows**

        step1: Setting enviroment for using MSVC.
        step2:
          \> cd slib++
          \> nmake -f makefile.win 

 * **linux**

          $ cd slib++
          $ make
