# **README for elibpp** #
***


## **What's elibpp?** ##
 * It's a C++ library of elib(evil-library).
 * It's a free and open source program base FreeBSD. Everyone can use it.



## **Building elibpp** ##
 * **Windows**

        step1: Setting enviroment for using MSVC.
        step2:
          \> cd elibpp
          \> nmake -f makefile.mk 

 * **Linux**

        $ cd elibpp
        $ make
