# **README for eutil**
***



## **What's eutil**
  * It's an utility C++(0x) library.
  * It's a free and open source program based FreeBSD.


## **Building eutil**
  * **Linux**

        $ cd eutil
        $ make

  * **Windows**

        Step1: Setting enviroment for using MSVC(VS2012+).
        Step2:
          \> cd eutil
          \> nmake -f Makefile.mk
