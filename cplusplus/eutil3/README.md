# **README for eutil**
***

## **What's eutil?**
  * It's an utility C++(0x) library.
  * It's a free and open source program based FreeBSD.

## **Building eutil**
  * **Linux**

        $ cd eutil
        $ make

  * **Windows**

        Step1: Setting environment for using MSVC(VC2012+).
        Step2:
          \> cd eutil
          \> name -f Makefile.win.mk
