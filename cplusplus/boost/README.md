## **Install**
    1) Download
        * http://sourceforge.net/projects/boost/files/boost/1.59.0/boost_1_59_0.7z/download
    2) Compile for Windows
        * Set MSVC environments
        * \> cd boost_1_59_0
        * \> bootstrap.bat
        * \> bjam.exe
    3) Compile for *nix
        * $ cd boost_1_59_0
        * $ sh bootstrap.sh
        * $ ./b2
