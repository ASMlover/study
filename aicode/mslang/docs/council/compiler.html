闭包与 Upvalue

Upvalue 捕获机制

当内层函数引用外层函数的局部变量时，Compiler 会创建一个 Upvalue 来捕获该变量。 编译期确定捕获方式：直接捕获局部变量，或传递外层 upvalue。

LOCAL

直接捕获外层局部变量

UPVALUE

传递外层函数的 upvalue

嵌套编译器结构

每个函数编译时创建新的 Compiler 实例，enclosing_ 指向外层编译器， 形成链表结构用于解析 upvalue。

Pratt Parser 可视化解析

parse_precedence 执行流程

1. 消费前缀

读取第一个 token，查找并执行对应的 prefix 处理函数

2. 检查优先级

比较下一个 token 的优先级与当前最小优先级

3. 处理中缀

如果优先级足够，消费 token 并执行 infix 处理

4. 循环或返回

继续检查直到遇到更低优先级的 token

左结合 vs 右结合

左结合 a - b - c

precedence + 1 → (a - b) - c

右结合 a = b = c

precedence (不加1) → a = (b = c)
