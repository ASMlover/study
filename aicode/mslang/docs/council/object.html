Object 对象系统

所有堆分配类型的基类，采用 C++ 继承体系实现多态。通过侵入式链表追踪所有对象， 支持字符串驻留、闭包、类实例等丰富类型。

对象类型层次

Object 基类

基类定义

class Object {
protected:
  ObjectType type_;       // 对象类型标签
  bool is_marked_ = false;  // GC 标记位
  Object* next_ = nullptr;   // 侵入式链表

public:
  explicit Object(ObjectType type) : type_(type) {}
  virtual ~Object() = default;

  // 每个子类必须实现
  virtual str_t stringify() const = 0;
  virtual void trace_references() {};
  virtual sz_t size() const = 0;

  // 访问器
  ObjectType type() const { return type_; }
  bool is_marked() const { return is_marked_; }
  void set_marked(bool m) { is_marked_ = m; }
  Object* next() const { return next_; }
  void set_next(Object* n) { next_ = n; }
};

侵入式对象链表

所有 GC 管理的对象通过 next_ 指针链接成链表， VM 通过 objects_ 链表头访问所有对象。

// 新对象插入链表头部
obj->set_next(objects_);
objects_ = obj;

核心对象类型详解

S

ObjString

字符串对象

• 不可变字符串
• 缓存 hash 用于快速比较
• 字符串驻留保证唯一性
F

ObjFunction

函数对象

• 存储编译后的字节码 (Chunk)
• 记录参数数量 (arity)
• 内联缓存优化属性访问
C

ObjClosure

闭包对象

• 包装函数 + 捕获的变量
• Upvalue 数组存储捕获值
• 运行时实际执行单位
U

ObjUpvalue

上值对象

• Open: location_ 指向栈
• Closed: 值移入 closed_
• 链表管理所有 open upvalue
K

ObjClass

类对象

• 方法表存储闭包
• Getter/Setter 支持
• 静态方法单独存储
I

ObjInstance

实例对象

• 指向所属类
• fields 存储实例属性
• 方法通过类查找

Upvalue 状态转换

Open 状态

变量仍在栈上，Upvalue 的 location_ 直接指向栈槽位。

location_ → &stack[slot]

Close 过渡

当变量离开作用域时，将值从栈复制到 closed 字段。

closed = *location_; location_ = &closed;

Closed 状态

变量已移至堆上，不再依赖栈，可被闭包长期引用。

location_ → &closed_ (self-reference)

方法调用与绑定

ObjBoundMethod

当访问实例方法时，创建一个绑定方法对象，将接收者 (receiver) 与方法闭包绑定。

class ObjBoundMethod : public Object {
Value receiver_; // 实例
ObjClosure* method_; // 方法闭包
};
调用过程
1. instance.method → 查找类方法表
2. 创建 ObjBoundMethod(instance, method)
3. 压入栈作为可调用值
4. OP_CALL 时 receiver 作为 slot[0]

This 绑定机制

当调用 bound method 时，receiver 自动作为第一个参数 (this/self) 传入

集合类型

ObjList - 动态数组

class ObjList : public Object {
  std::vector<Value> elements_;
public:
  void trace_references() {
    for (auto& v : elements_)
      mark_value(v);
  }
};
O(1) 索引 动态扩容 可变长

ObjMap - 哈希映射

class ObjMap : public Object {
  ValueMap entries_;  // unordered_map<Value, Value>
public:
  void trace_references() {
    for (auto& [k, v] : entries_) {
      mark_value(k);
      mark_value(v);
    }
  }
};
O(1) 查找 任意键类型 Value 哈希
