核心实现详解

1. 空白字符处理

Scanner 首先调用 skip_whitespace() 跳过所有空白字符（空格、制表符、换行）。 同时处理注释：// 开头的行注释会被跳过。

// 跳过空白和注释
while (true) {
char c = peek();
switch (c) {
case ' ':
case '\t':
case '\r':
advance(); break;
case '\n':
line_++; advance(); break;
case '/':
if (peek_next() == '/')
skip_line_comment();
}
}

2. 关键字识别 - Trie 结构

识别标识符后，使用类似 Trie 的结构检查是否是关键字。 通过 check_keyword() 逐步匹配字符， 避免字符串比较开销。

TokenType identifier_type() {
switch (start_[0]) {
case 'a': return check_keyword(1, 2, "nd", TOKEN_AND);
case 'c':
if (current_ - start_ > 1) {
switch (start_[1]) {
case 'l': return check_keyword(2, 3, "ass", TOKEN_CLASS);
case 'a': return check_keyword(2, 2, "se", TOKEN_CASE);
}
}
break;
}
return TOKEN_IDENTIFIER;
}

3. 字符串插值处理

Maple 支持字符串插值语法 "Hello, ${name}!"。 Scanner 使用栈来追踪嵌套的插值大括号层级。

// 插值语法解析
if (c == '$' && peek_next() == '{') {
interp_braces_[interp_count_++] = 1;
advance(); advance();
return make_token(TOKEN_INTERP_BEGIN);
}
// 追踪嵌套大括号
if (c == '{') {
interp_braces_[interp_count_-1]++;
} else if (c == '}') {
if (--interp_braces_[interp_count_-1] == 0) {
interp_count_--;
return make_token(TOKEN_INTERP_END);
}
}
