## **Plugins**
> ### **grep.vim**
    1) 将grep.vim拷贝到~/.vim/plugin
    2) 在.vimrc中设置映射快捷键
        nnoremap <silent> <C-F3> :Grep<CR>
    3) 常用命令
        * :Grep 在当前打开的文件中搜寻
        * :Rgrep 在目录中递归搜寻
        * :GrepBuff 在Vim的buff文件中搜寻
